Traffic the Go web framework

Easy to use web framework for Go.

Features

  • Regexp routing
  • Before Filters
  • Custom not found handler
  • Middlewares

Installation

$ go get github.com/pilu

Usage

router := traffic.New()

// Routes
router.Get("/", rootHandler)
router.Get("/categories/:category_id/pages/:id", pageHandler)

http.Handle("/", router)
http.ListenAndServe(":7000", nil)
